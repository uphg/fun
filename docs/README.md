使用 npm 安装

```sh
npm install @uphg/fun
# or yarn add @uphg/fun
```

在项目中引入

```js
// ES6 modules
import each from 'uphg/fun/each'

// CommonJS modules
const each = require('@uphg/fun/each')
```
